Description

Ce script permet d'effectuer une rotation autour d'un centre donné.
La méthode pour définir une rotation est la suivante :

Rotate(float angle [, float x [, float y]])

angle : angle en degrés.
x : abscisse du centre de rotation. Valeur par défaut : position courante.
y : ordonnée du centre de rotation. Valeur par défaut : position courante.

La rotation affecte tous les éléments qui sont dessinés après avoir appelé la méthode (à l'exception des zones cliquables).

Remarques :

- Seul l'affichage est modifié. Les méthodes GetX() et GetY() ne sont pas affectées, ni le mécanisme de saut de page automatique.
- La rotation n'est pas conservée de page en page. Chaque page commence avec une rotation nulle.
